
# Пример MIP-формулировки задачи для 2 героев и 3 мельниц

## Условие примера

Есть:

- таверна `0`
- три мельницы:
  - `1`
  - `2`
  - `3`
- два героя:
  - герой `1`
  - герой `2`

Таверна **не является** обычной вершиной маршрута в модели.  
Она учитывается отдельно через **стоимость старта**. Это более приближено к исходной формулировке задачи, хотя и делает уравнения несколько сложнее.

## Обозначения

### Ёмкости героев

$c_1,\; c_2$

### Расстояния от таверны

$s_1,\; s_2,\; s_3$

где:
- $s_1$ — расстояние от таверны до мельницы 1
- $s_2$ — расстояние от таверны до мельницы 2
- $s_3$ — расстояние от таверны до мельницы 3

### Расстояния между мельницами

Для общности пишем расстояния по дугам:

$d_{12}, d_{13}, d_{21}, d_{23}, d_{31}, d_{32}$

Так как расстояние между мельницами не зависит от направления движения, то:

$d_{12} = d_{21}, \quad d_{13}=d_{31}, \quad d_{23}=d_{32}$


### Стоимость посещения

$V_c = 100$

## Таблица переменных

### 1. Переменные модели

| Обозначение | Тип | Смысл |
|---|---:|---|
| $a_k$ | бинарная | герой $k$ активен |
| $v_{i,k}$ | бинарная | герой $k$ посещает мельницу $i$ |
| $x_{i,j,k}$ | бинарная | герой $k$ идёт по дуге $i \to j$ |
| $u_{i,k}$ | непрерывная | служебная MTZ-переменная порядка |

### 2. Какие переменные появляются в этом примере

#### Активность героев

$a_1,\; a_2$

#### Посещение мельниц

Для героя 1:

$v_{1,1},\; v_{2,1},\; v_{3,1}$

Для героя 2:

$v_{1,2},\; v_{2,2},\; v_{3,2}$

#### Дуги маршрута

Для каждого героя возможны 6 направленных дуг:

$1\to2,\;1\to3,\;2\to1,\;2\to3,\;3\to1,\;3\to2$

То есть, например, для героя 1:

$x_{1,2,1},\;x_{1,3,1},\;x_{2,1,1},\;x_{2,3,1},\;x_{3,1,1},\;x_{3,2,1}$

И аналогично для героя 2.

#### MTZ-переменные

Для героя 1:

$u_{1,1},\;u_{2,1},\;u_{3,1}$

Для героя 2:

$u_{1,2},\;u_{2,2},\;u_{3,2}$

## Целевая функция

Мы хотим:

1. максимизировать число посещённых мельниц;
2. при равенстве — предпочесть решение с большим leftover.

Поэтому целевая функция выглядит так:

$$
\max
\Bigl(
\sum_{i=1}^{3}\sum_{k=1}^{2} v_{i,k}
+
\varepsilon (L_1 + L_2)
\Bigr)
$$

где:
- $L_1$ — leftover героя 1,
- $L_2$ — leftover героя 2,
- $\varepsilon$ — очень маленький коэффициент.

## Таблица выражений используемых в уравнениях

| Выражение | Смысл |
|---|---|
| $\sum_i v_{i,k}$ | сколько объектов посетил герой $k$ |
| $\sum_p x_{p,i,k}$ | сколько дуг входит в объект $i$ у героя $k$ |
| $\sum_s x_{i,s,k}$ | сколько дуг выходит из объекта $i$ у героя $k$ |
| $v_{i,k} - \sum_p x_{p,i,k}$ | объект $i$ является стартом маршрута героя $k$ |
| $v_{i,k} - \sum_s x_{i,s,k}$ | объект $i$ является концом маршрута героя $k$ |
| $L_k$ | leftover героя $k$ |

> Почему выражение $v_{i,k} — \sum_p x_{p,i,k}$ показываетя является ли объект $i$ является стартом маршрута героя $k$?

Потому что $v_{i,k}$ показывает посетил ли герой мельницу, а выражение $\sum_p x_{p,i,k}$ показывает есть ли входящие дуги (таверна в нашей модели не считается стартом маршрута и мы не считаем дугой путь из таверны в стартовую мельницу).

Получается что выражение равняется 1, если $v_{i,k}$ равняется 1 (т.е. герой посетил мельницу), а $\sum_p x_{p,i,k}$ равняется 0 (т.е. входящих дуг нет).

Аналогичным образом мы определяем что объект $i$ является концом маршрута героя $k$.

## Стоимость маршрута героя

### Герой 1

$$
used_1 =
s_1\Bigl(v_{1,1}-(x_{2,1,1}+x_{3,1,1})\Bigr)
+
s_2\Bigl(v_{2,1}-(x_{1,2,1}+x_{3,2,1})\Bigr)
+
s_3\Bigl(v_{3,1}-(x_{1,3,1}+x_{2,3,1})\Bigr)
$$

$$
\quad +\;
d_{12}x_{1,2,1}+d_{13}x_{1,3,1}
+d_{21}x_{2,1,1}+d_{23}x_{2,3,1}
+d_{31}x_{3,1,1}+d_{32}x_{3,2,1}
$$

$$
\quad +\;
V_c(v_{1,1}+v_{2,1}+v_{3,1})
-
(V_c-1)a_1
$$

$v_{1,1}-(x_{2,1,1}+x_{3,1,1})$ — показывает посетил ли первый герой первую мельницу. Соответственно $s_1\Bigl(v_{1,1}-(x_{2,1,1}+x_{3,1,1})\Bigr)$ — это стоимость старта из таверны в первую мельницу.

$V_c(v_{1,1}+v_{2,1}+v_{3,1})$ — стоимость всех посещенных мельниц.

$(V_c-1)a_1$ — пытаемся учесть правило последнего хода. Мы всегда вычитаем 99 ($V_c-1$), даже если правило последнего хода не использовалось для данного героя.

### Герой 2

$$
used_2 =
s_1\Bigl(v_{1,2}-(x_{2,1,2}+x_{3,1,2})\Bigr)
+
s_2\Bigl(v_{2,2}-(x_{1,2,2}+x_{3,2,2})\Bigr)
+
s_3\Bigl(v_{3,2}-(x_{1,3,2}+x_{2,3,2})\Bigr)
$$

$$
\quad +\;
d_{12}x_{1,2,2}+d_{13}x_{1,3,2}
+d_{21}x_{2,1,2}+d_{23}x_{2,3,2}
+d_{31}x_{3,1,2}+d_{32}x_{3,2,2}
$$

$$
\quad +\;
V_c(v_{1,2}+v_{2,2}+v_{3,2})
-
(V_c-1)a_2
$$

## Leftover

$L_1 = c_1 a_1 - used_1$

$L_2 = c_2 a_2 - used_2$

## Ограничения

### 1. Каждую мельницу можно посетить не более одного раза

$v_{1,1}+v_{1,2} \le 1$

$v_{2,1}+v_{2,2} \le 1$

$v_{3,1}+v_{3,2} \le 1$

### 2. Если герой не активен, он не посещает объекты

У каждого героя максимум 3 объекта:

$v_{1,1}+v_{2,1}+v_{3,1} \le 3a_1$

$v_{1,2}+v_{2,2}+v_{3,2} \le 3a_2$

### 3. У активного героя ровно один старт

#### Герой 1

$$
\bigl(v_{1,1}-(x_{2,1,1}+x_{3,1,1})\bigr)
+
\bigl(v_{2,1}-(x_{1,2,1}+x_{3,2,1})\bigr)
+
\bigl(v_{3,1}-(x_{1,3,1}+x_{2,3,1})\bigr)
= a_1
$$

#### Герой 2

$$
\bigl(v_{1,2}-(x_{2,1,2}+x_{3,1,2})\bigr)
+
\bigl(v_{2,2}-(x_{1,2,2}+x_{3,2,2})\bigr)
+
\bigl(v_{3,2}-(x_{1,3,2}+x_{2,3,2})\bigr)
= a_2
$$

### 4. У активного героя ровно один конец

#### Герой 1

$$
\bigl(v_{1,1}-(x_{1,2,1}+x_{1,3,1})\bigr)
+
\bigl(v_{2,1}-(x_{2,1,1}+x_{2,3,1})\bigr)
+
\bigl(v_{3,1}-(x_{3,1,1}+x_{3,2,1})\bigr)
= a_1
$$

#### Герой 2

$$
\bigl(v_{1,2}-(x_{1,2,2}+x_{1,3,2})\bigr)
+
\bigl(v_{2,2}-(x_{2,1,2}+x_{2,3,2})\bigr)
+
\bigl(v_{3,2}-(x_{3,1,2}+x_{3,2,2})\bigr)
= a_2
$$

### 5. Вход в объект возможен только если объект посещён

#### Герой 1

$x_{2,1,1}+x_{3,1,1} \le v_{1,1}$

$x_{1,2,1}+x_{3,2,1} \le v_{2,1}$

$x_{1,3,1}+x_{2,3,1} \le v_{3,1}$

#### Герой 2

$x_{2,1,2}+x_{3,1,2} \le v_{1,2}$

$x_{1,2,2}+x_{3,2,2} \le v_{2,2}$

$x_{1,3,2}+x_{2,3,2} \le v_{3,2}$

### 6. Выход из объекта возможен только если объект посещён

#### Герой 1

$x_{1,2,1}+x_{1,3,1} \le v_{1,1}$

$x_{2,1,1}+x_{2,3,1} \le v_{2,1}$

$x_{3,1,1}+x_{3,2,1} \le v_{3,1}$

#### Герой 2

$x_{1,2,2}+x_{1,3,2} \le v_{1,2}$

$x_{2,1,2}+x_{2,3,2} \le v_{2,2}$

$x_{3,1,2}+x_{3,2,2} \le v_{3,2}$

### 7. Запрет недопустимого старта

Пусть:
- $S_1$ — допустимые стартовые объекты героя 1
- $S_2$ — допустимые стартовые объекты героя 2

Если объект $i$ не может быть стартом героя $k$, то:

$v_{i,k} \le \sum_{p \ne i} x_{p,i,k}$

То есть если герой посещает этот объект, в него обязательно должна входить дуга.

### 8. Ограничения по очкам хода

#### Герой 1

$used_1 \le c_1$

#### Герой 2

$used_2 \le c_2$

### 9. MTZ-ограничения

Так как здесь 3 объекта, для каждого героя есть 6 MTZ-ограничений — по одному на каждую дугу.

#### Герой 1

$u_{1,1}-u_{2,1}+3x_{1,2,1}\le 2$

$u_{1,1}-u_{3,1}+3x_{1,3,1}\le 2$

$u_{2,1}-u_{1,1}+3x_{2,1,1}\le 2$

$u_{2,1}-u_{3,1}+3x_{2,3,1}\le 2$

$u_{3,1}-u_{1,1}+3x_{3,1,1}\le 2$

$u_{3,1}-u_{2,1}+3x_{3,2,1}\le 2$

#### Герой 2

$u_{1,2}-u_{2,2}+3x_{1,2,2}\le 2$

$u_{1,2}-u_{3,2}+3x_{1,3,2}\le 2$

$u_{2,2}-u_{1,2}+3x_{2,1,2}\le 2$

$u_{2,2}-u_{3,2}+3x_{2,3,2}\le 2$

$u_{3,2}-u_{1,2}+3x_{3,1,2}\le 2$

$u_{3,2}-u_{2,2}+3x_{3,2,2}\le 2$

### 10. Границы на MTZ-переменные

#### Герой 1

$u_{1,1} \ge v_{1,1}, \qquad u_{1,1} \le 3v_{1,1}$

$u_{2,1} \ge v_{2,1}, \qquad u_{2,1} \le 3v_{2,1}$

$u_{3,1} \ge v_{3,1}, \qquad u_{3,1} \le 3v_{3,1}$

#### Герой 2

$u_{1,2} \ge v_{1,2}, \qquad u_{1,2} \le 3v_{1,2}$

$u_{2,2} \ge v_{2,2}, \qquad u_{2,2} \le 3v_{2,2}$

$u_{3,2} \ge v_{3,2}, \qquad u_{3,2} \le 3v_{3,2}$

## Таблица «переменная → ограничение → смысл»

| Что | Формула | Смысл |
|---|---|---|
| Активность героя | $a_k$ | герой используется или нет |
| Посещение объекта | $v_{i,k}$ | герой $k$ посещает объект $i$ |
| Переход по дуге | $x_{i,j,k}$ | герой идёт из $i$ в $j$ |
| Порядок в маршруте | $u_{i,k}$ | техническая переменная для запрета циклов |
| Объект не более 1 раза | $\sum_k v_{i,k}\le1$ | один объект не может быть у двух героев |
| Связь с активностью | $\sum_i v_{i,k}\le 3a_k$ | неактивный герой не посещает ничего |
| Один старт | $\sum_i(v_{i,k}-in_{i,k})=a_k$ | у активного героя ровно один старт |
| Один конец | $\sum_i(v_{i,k}-out_{i,k})=a_k$ | у активного героя ровно один конец |
| Вход только в посещённый объект | $in_{i,k}\le v_{i,k}$ | если объект не посещён, в него нельзя входить |
| Выход только из посещённого объекта | $out_{i,k}\le v_{i,k}$ | если объект не посещён, из него нельзя выходить |
| Запрет плохого старта | $v_{i,k}\le in_{i,k}$ для $i\notin S_k$ | объект не может быть первым |
| Ограничение по ходу | $used_k\le c_k$ | герой не должен тратить больше хода, чем имеет |
| MTZ | $u_{i,k}-u_{j,k}+3x_{i,j,k}\le2$ | запрет подциклов |
